### Frontend

In frontend I used:

- HTML
- CSS
- Javascript
- SASS
- Handlebars
- BEM Architecture
- SPA system
- Responsive design / mobile first

### Backend


In Backend I used:

- Node.JS
- Express
- Mongoose
- MongoDB
- Layer architecture to improve scalability and fast learning
